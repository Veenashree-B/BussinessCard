# Digital Business Card App 📱

A sleek, modern Android application built to serve as an interactive digital business card. This project demonstrates a strong understanding of modern Android development practices, specifically focusing on building declarative user interfaces.

## ✨ Features
* **Modern UI:** Built entirely with **Jetpack Compose** following Material Design 3 guidelines for a crisp, professional look.
* **Component-Based Architecture:** Demonstrates clean code practices by breaking down the UI into modular, reusable components (e.g., Profile Section, Contact Section).
* **Responsive Layout:** Elegantly handles spacing and alignment to ensure a great user experience on various screen sizes.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Design System:** Material 3
* **Build System:** Gradle
